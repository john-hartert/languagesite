The first two weeks covered basic terminal commands, and the use of python. A variety of python basics were discussed. Including: the use of the raw_input function, if/else statements, and loops. Useage of for loops and while loops were covered extensively.

Projects

  1. RPG Games using while loops and if/else statements.
  2. An application which tracks lunch visits. Concepts incorporated included File I/O for saving and restoring entries and a built in graph function.
  3. A phonebook which also incorporated File I/O for saving and restoring saved entries. The software could also receive new entries, delete old entries and display all entries.
